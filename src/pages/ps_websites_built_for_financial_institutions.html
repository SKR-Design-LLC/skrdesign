ProfitStars

Building Websites for Banking & Financial Institutions

Advanced Web Builder at ProfitStars from January 2014 to April 2019. Our department, Web Solutions, provide website design, responsively built websites, hosting, and content management for banking and financial institutions.

The designer provided Illustrator files of the site mockups and I built the site in HTML, CSS and JavaScript with Foundation or Bootstrap base. When building website I was also responsible for building to ADA standards, SEO optimize, cross browser and device testing. Once tested and completed website are added to a content management system within project timelines.
