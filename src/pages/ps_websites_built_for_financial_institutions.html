ProfitStars

Building Websites for Banking & Financial Institutions

Advanced Web Developer at ProfitStars, a Jack Henry & Associates Company, in the Fintech industy for five years. The Web Solutions department provided website design, responsively and ADA compliant built websites, hosting, and content management for banking and financial institutions. Each site was assigned a project team comprised of a project coordinator, designer and developer.

The designer provided the website developer Adobe Illustrator files of the site mockups and wireframes. I would then use them to build the site in HTML, CSS and JavaScript with a Bootstrap base to pixel perfectly match the mockups. Responsible for building to ADA standards, SEO optimize, cross browser and device testing. Once tested and completed the website were added to a content management system within project timelines.

The sites shown are the property of the bank.
