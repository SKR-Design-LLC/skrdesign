Chapter Two

ProfitStars

Building Websites for Banking & Financial Institutions

Advanced Web Builder at ProfitStars from January 2014 to April 2019. Our department, Web Solutions, provide website design, responsively and ADA compliant built websites, hosting, and content management for banking and financial institutions. We work both in a team and individually on our website projects. Each site was assigned a team; a project coordinator, designer and builder. The project coordinator worked with the client and the team to set deadlines for tasks, starting with the design of the site.

Once the designer had completed their tasks and the it was approved by the client, the website builder (myself) was provided Adobe Illustrator files of the site mockups and would then built the site in HTML, CSS and JavaScript with either a Foundation or Bootstrap base. When building website I was also responsible for building to ADA standards, SEO optimize, cross browser and device testing. Once tested and completed the website are added to a content management system within project timelines.

The sites shown are the property of the bank.
